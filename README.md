# argo-container
A batteries included approach to get started with Urban Informatics / Civic Data science


A docker container that includes:

- anaconda
- boto
- jq
- csvkit
- opencv
- Linux Desktop
- QGIS
- R Shiny
- PLotly + DASH
- Flask
- aws cli / aws-shell
- scikit-learn
- pdfgrep
- data science toolkit
- geopandas
- osmnx
